Android organizes its UI elements into layouts and views. Everything you see, such as a button, label, or text box, is a <android.view.View> object. Layouts organize views, such as grouping together a button and label or a group of these elements. As we are about to discover, Android provides different layouts, each with its own way of organizing its children. Each type of layout is a specialized <android.view.ViewGroup> object.

Consider this....

A layout can contain other children. And each of those children can furthermore be layouts themselves

This means.... as a savvy Android developer you can use any number, any combination of layouts and/or views, allowing you to create very exciting user interfaces.

See https://developer.android.com/guide/topics/ui/overview.html
