Let's create an app to further study how to handle events in Android.

The app we'll create is named: Big Button App, and you're encouraged to build your own version.

Android Studio Application

Launch Android Studio, and Start a new Android Studio project.

Configure your new project

  • Application Name: Big Button First App
  • Company Domain: yourCollegeUsername.algonquincollege.com

 

Select the form factors your app will run on

  • Select: Phone and Table
  • Minimum SDK: API 28: Android 9.0 (Pie)

 

Add an activity to Mobile

  • Select: Empty Activity

 

Customize the Activity

  • Click the Finish button to accept the default settings.

 

Add an Externalized String

Edit strings.xml (res > values), and add following externalized string for the button's text:

<string name="buttonText">Big Button</string>

Save your changes.

 

Replace the Constraint Layout

Edit activity_main.xml (res > layout), switch to Text pane, select all of the XML code and delete it.

Next, replace the content with the following:

<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <Button
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_centerHorizontal="true"
        android:layout_centerVertical="true"
        android:text="@string/buttonText"
        android:textSize="36sp" />

</RelativeLayout>

Run the App

Run Big Button App on the Android emulator.
