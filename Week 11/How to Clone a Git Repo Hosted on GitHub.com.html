  • To clone a git repo hosted on GitHub.com

Clone a Project from GitHub

Launch Android Studio
Close all open projects (File > Close Project). Repeat until you see the Welcome to Android Studio dialog.
Check out project from Version Control
Choose: GitHub. Sign-in to your github.com account (if prompted).
Clone Repository
For example, suppose you wish to clone my version of ActivityLifecyle. To clone my git repo, enter the following values:
    • Git Repository URL: https://github.com/hurdleg/ActivityLifecyle.git
    • Parent Directory: Important: the name of my project is ActivityLifecyle, and is the same name as your project. Click the "..." and create a folder named: GitHub (and choose this folder). This will prevent my project from over-writing (i.e. clobber) your project :)
    • Directory Name: ActivityLifecyle

and click Clone

Checkout From Version Control
Click Yes to open the project.
If you see this error message: Error Loading Project click Details... and then click Remove Selected. Fear not.... this error message goes away when you re-build the project.
ActivityLifecycle Project
Perform a fresh Build of the project: Build > Make Project
Gradle Console
Open the Gradle Console window (View > Tool Windows). Scroll to the bottom and verify you see: BUILD SUCCESSFUL. If seen, continue. If not seen, you need some help.
Run
Run > Run 'app' on the CST8227 AVD.

Enjoy!
