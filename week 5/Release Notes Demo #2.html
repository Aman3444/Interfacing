Software PDL

Declarations

  • SER
  • LATCH
  • CLOCK

Setup

  • set the pin mode of SER, LATCH, and CLOCK as output
  • initialize SER, LATCH, CLOCK to 0 (digitalWrite())

Loop

int bitArray[] = {1, 3, 7, 15, 31, 63, 127, 255};

//count up first 8-bits

for i = 0 to 8:

//shift first 8 bits; ends up in the second SR

shift out 0 (mode: MSBFIRST)

//shift second 8-bits; ends up in the first SR

shift out bitArray[i] (mode: MSBFIRST)

toggle the LATCH (write HIGH, then write LOW)

delay

//count up second 8-bits

for i = 0 to 8:

//shift first 8 bits; ends up in the second SR

shift out bitArray[i] (mode: MSBFIRST)

//shift second 8-bits; ends up in the first SR

shift out 255 (mode: MSBFIRST)

toggle the LATCH (write HIGH, then write LOW)

delay

//count down second 8-bits

for i = 7 to 0:

//shift first 8 bits; ends up in the second SR

shift out bitArray[i] (mode: MSBFIRST)

//shift second 8-bits; ends up in the first SR

shift out 255 (mode: MSBFIRST)

toggle the LATCH (write HIGH, then write LOW)

delay
