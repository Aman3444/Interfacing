Before You Begin

This lab is based on Building Your First App on the developer.android.com training website, but with noticeable differences.

From the website, read the following sections:

  • Creating an Android Project
  • Run Your App
  • Building a Simple User Interface

Using Building Your First App as a guide, make your own first app as specified below.

Notable differences (please read):

  • use API 28
  • use the CST8227 AVD

Android Studio Application

Launch Android Studio, and Start a new Android Studio project.

Configure your new project

  • Application Name: My First App
  • Company Domain: yourCollegeUsername.algonquincollege.com

 

  1. Notice the genereated Package name: com.algonquincollege.yourCollegeUsername.myfirstapp
  2. Make note of the Project location on your file-system

Select the form factors your app will run on

  • Select: Phone and Table
  • Minimum SDK: API 28: Android 9.0 (Pie)

Add an activity to Mobile

  • Select: Empty Activity

Customize the Activity

  • Click the Finish button to accept the default settings.
