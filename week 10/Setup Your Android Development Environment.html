Android developers can choose from different programming languages to make Android apps. For example, you can make Android apps using HTML5, Javascript, and CSS. You can also choose to develop your Android apps in kotlin (https://kotlinlang.org/).

In CST8227, we'll developer our Android apps in Java.

Get the Java JDK

Already have the Java JDK installed on your laptop? Yes - skip this step, and proceed to the next step.

The Java programming language is used to create native Android applications.

For this course, you are instructed to use this version of the Java Development Kit (JDK): Java Standard Edition (SE) 8u191 / 8u142

Notes:

  • we're Android developers, so we require the entire JDK. The Java Runtime Edition (JRE) cannot be used to develop Java applications nor Android apps
  • the JDK includes the JRE (but not vice-versa; see the previous statement :)

Already have the Java JDK?

If you've already downloaded and installed the Java JDK on your computer, perform an update to the Java JDK.

Update your version of the Java JDK to Java 8 Update 1.91

The Android Integrated Development Environment (IDE)

According to http://developer.android.com/studio

Android Studio is the official Integrated Development Environment (IDE) for Android app development

Download the latest version (*) of Android Studio for your preferred operating system (remember, Mac for me). Make sure the Android SDK tools are bundled too; this is the default behaviour.

* Already have Android Studio (AS) installed? Then launch AS, and update to version 3.2.1
