1.0 Prevent Future Updates

It's critical that the entire class --- you and me --- run the exact same versions of the entire Android software stack.

We are using version 3.2.1

Prevent Android Studio from performing future updates:

Android Studio > Check for Updates... > click the Updates link

  • Un-check: Automatically check updates for Android SDK
Pro-Tip: remember to enable this once this course finishes. 

2.0 Appearance

Would you like to change the entire appearance of Android Studio? I did :)

Android Studio > Preferences... > Appearance & Behavior > Appearance

  • Theme: <select a theme>
Pro-Tip: you can download new themes from http://ideacolorthemes.org Once you have downloaded the .jar file, go to File -> Import Settings... and choose the file downloaded. For more information, see http://stackoverflow.com/questions/16579448/how-to-change-or-add-theme-to-android-studio

3.0 Editor: Show line numbers

Do you like to see the line numbers in the editor? I do :)

Android Studio > Preferences... > Editor > General > Appearance

  • Check: Show line numbers
