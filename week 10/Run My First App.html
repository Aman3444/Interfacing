Run My First App

At this point, you should be able to run your My First App within the Android emulator.

First, build your project: Build > Make Project

Examine the output generated by Gradle (View > Tool Windows > Gradle Console), and look for: BUILD SUCCESSFUL

Now run your app: Run > Run 'app'

If the Choose Device dialog appears, select Launch emulator and select the MAD9132 AVD.

Enjoy!
