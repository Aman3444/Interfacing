Q: My Android Studio project is messed up, and now my project won't build. Can I edit / delete the R.java in hopes that my project will build?

A: The short answer: no, do not edit / delete the R.java file.

This file is auto-generated by AAPT (Android Asset Packaging Tool), and contains the resource IDs for all the resources of the res/ directory.

If you create any component in the activity_main.xml file, the ID for the corresponding component is automatically created in this file.

This ID can be used in the activity source file to perform any action on the component.
