Explore Your Android Studio Project

Open the Project window: View > Tool Windows > Project

And the Android view is selected from the drop-down list at the top of that window.

You can then see the following files:

app > java > com.algonquincollege.hurdleg.myfirstapp > MainActivity.java: This is the main activity (the entry point of your app). When you build and run the app, the system launches an instace of this Activity and loads its layout.

gen [Generated Java Files] -> R.java: the generated R values; links the layout XML files to the Activity source files in src. Do not modify the contents of this folder.

res > layout > activity_main.xml: This XML file defines the layout for the activity's UI. It contains a TextView element with the text "Hello World!".

res > values > strings.xml: defines the app's string resources.

manifests > AndroidManifest.xml: The manifest file describes the fundamental characteristics of the app and defines each of its components. Expressed in a different way: defines the meta information for the Android project.

Gradle Scripts > build.gradle: You'll see two files with this name: one for the project and one for the "app" module. Each module has its own build.gradle file, but this project currenty has just one module. You'll mostly work with the module's build.gradle file to configure how the Gradle tools compile and build your app. For more information about this file, see Configure Your Build

Please read this additional material: http://developer.android.com/training/basics/firstapp/creating-project.html
