Android's ImageView

Would you like to display an image in your Android app?

Good news... Android includes a widget named ImageView to display images.

Before You Begin

Please read Android's API documentation for class ImageView

https://developer.android.com/reference/android/widget/ImageView.html

How to Use an ImageView

Let's create an Android Studio project featuring ImageView.

Name your project: Demo ImageView

On the Customize the Activity screen, un-check Backward Compatibility (AppCompat).

Import an Image into Android Studio

Next, import your favourite image into Android Studio. ImageView supports popular image formats, including PNG and JPEG.

The steps:

  1. Download your favourite image to your Desktop. Copy your image into your operating system's clipboard. On Mac, command-c (control-c on Windows)
  2. Paste your image into the drawable resource folder (res > drawable). On Mac, command-v (control-v on Windows)

 

The View

Edit activity_main.xml (res > layout),  and replace the XML content with:

<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <ImageView
        android:id="@+id/imageView"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:layout_centerHorizontal="true"
        android:layout_centerVertical="true"
        android:contentDescription="Duke - the mascot of Java"
        android:src="@drawable/Duke_java_maskot" />

</RelativeLayout>

Notes:

  • the (default) ConstraintLayout was replaced with a RelativeLayout
  • the height & width of the ImageView is set to: match_parent. The parent of ImageView is RelativeLayout
  • the android:src property is set to: @drawable/Duke_java_maskot
  • the android:contentDescription is set to the literal string: "Duke - the mascot of Java". This string should be externalized.
